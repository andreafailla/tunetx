# Glossary

## note

A note is a single musical pitch, such as C or G sharp.

## note group

A note group is a small collection of notes treated as one object.

## pitch class

A pitch class is a note identity without octave. In plain English, C4 and C5 both count as C.

## pitch-class set

A pitch-class set is a note group where octave is ignored and the notes are reduced to pitch classes.

## row

A row is an ordered sequence of pitch classes where order matters.

## rhythm sequence

A rhythm sequence is an ordered list of note lengths.

## duration token

A duration token is a short text label for a note length, such as `q` for quarter note or `e` for eighth note.

## interval vector

An interval vector is a count of the interval content inside a note group.

## normal order

Normal order is the most compact cyclic arrangement of a note group.

## prime form

Prime form is a normalized version of a note group used for comparison.

## voice leading

Voice leading is the small note-by-note motion that turns one note group into another.

## operator

An operator is a compact label that describes a transformation.

## graph

A graph is a map of relationships made of nodes and edges.

## node

A node is one item in a graph, such as one note group or rhythm pattern.

## edge

An edge is a connection between two nodes.

## score slice

A score slice is a group of notes that sound at the same time in a parsed score.

## MIDI

MIDI is a standard format for representing notes, timing, and performance data.

## sonification

Sonification is the process of turning non-audio data into sound.

## descriptor

A descriptor is a compact numeric summary used to compare musical objects.

## timbre

Timbre is the character or tone color of a sound, separate from pitch and timing.
